# AGENTS.md

## Project purpose
This repository hosts a public educational site for a child and adolescent psychiatry consult service.

## Architecture priorities
- Organize primarily by topic.
- Use topic pages as the main learner-facing destinations.
- Use Clinical Pearl pages for reusable cross-cutting practical references.
- Treat archived teaching as a resource type embedded within topic pages, not as a standalone section.
- Keep the site easy for one maintainer to update.

## Technical preferences
- Prefer Eleventy-native patterns.
- Use front matter, collections, and directory/global data cleanly.
- Avoid unnecessary frameworks or complexity.
- Keep file names and URLs clean and predictable.

## UX priorities
- Mobile-friendly.
- Clean and minimal.
- Fast to browse during clinical work.
- Searchable and easy to scan.

## Working style
- Plan first for structural changes.
- Explain major tradeoffs before implementing them.
- Preserve reusable metadata and avoid duplication.
